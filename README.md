# HexMetal
Toolhead carriage designed around SLM printing and use in the Hex Zero printer for ants or any other front facing MGN9H rail printers.
While it's made from scratch this is heavily inspired by the already incredible work Alex has done designing the Hex Zero.
Go and get yourself an awesome ant-sized printer: https://github.com/Alexander-T-Moss/Hex-Zero/tree/main

The .STL and .STEP files for the carriage, along with input shaping results will be uploaded once this is tested. Currently awaiting the first prototype by PCBWay :)

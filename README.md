# HexMetal
This is a from scratch design for an X Carriage to be SLM printed and used in the Hex Zero printer for ants.
While it's made from scratch this is heavily inspired by the already incredible work Alex has done designing the Hex Zero.
Go and get yourself an ant-sized printer: https://github.com/Alexander-T-Moss/Hex-Zero/tree/main

The .STL and .STEP files for the carriage, along with resonance testing results will be uploaded once this is tested. Currently awaiting the first prototype by PCBWay :)
